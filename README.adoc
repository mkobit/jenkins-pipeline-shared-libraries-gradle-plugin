== Purpose

A Gradle plugin for developing and testing link:{uri-jenkins-shared-library-docs}[Jenkins Pipeline Shared Libraries].

.Features
* Groovy compilation of `src/` and `vars/` against Jenkins core and plugin APIs
* Unit testing with link:{uri-jenkins-pipeline-unit}[Jenkins Pipeline Unit]
* Integration testing with link:{uri-jenkins-test-harness}[Jenkins Test Harness] (`JenkinsRule`)
* `@Grab` / `@NonCPS` support in shared library source
* Source code generation for `LocalLibraryRetriever` (wires local library into `JenkinsRule` tests)
* IDE code completion via GDSL

== Quick start

[source,kotlin]
----
// settings.gradle.kts
pluginManagement {
    repositories {
        gradlePluginPortal()
    }
}

dependencyResolutionManagement {
    repositories {
        mavenCentral()
        maven("https://repo.jenkins-ci.org/public/")
    }
}
----

[source,kotlin]
----
// build.gradle.kts
plugins {
    id("com.mkobit.jenkins.pipelines.shared-library") version "0.11.0"
}
----

The plugin applies the Groovy plugin, wires Jenkins core and the default workflow plugins onto the compile classpath, and sets up `test` (JPU) and `integrationTest` (JenkinsRule) suites.
The Jenkins BOM for the configured LTS line is injected automatically.

=== Overriding Jenkins versions

[source,kotlin]
----
sharedLibrary {
    jenkins {
        version = "2.492.3"
        testHarnessVersion = "2397.v5e2b_42e5e01c"
        bomVersion = "3463.v23b_7bb_b_b_66d5"
    }
}
----

=== Adding plugins

[source,kotlin]
----
dependencies {
    // version supplied by the auto-injected BOM
    jenkinsPlugin("org.jenkins-ci.plugins:git")
    jenkinsPlugin("org.jenkins-ci.plugins.workflow:workflow-aggregator")
}
----

== Writing integration tests

Integration tests live in `test/integration/` and run against an embedded Jenkins instance via `JenkinsRule`.
The plugin generates `LocalLibraryRetriever` so the local library is available inside `JenkinsRule` without a network checkout.

[source,groovy]
----
// test/integration/groovy/com/example/MyStepTest.groovy
import com.mkobit.jenkins.pipelines.codegen.LocalLibraryRetriever
import org.jenkinsci.plugins.workflow.cps.CpsFlowDefinition
import org.jenkinsci.plugins.workflow.job.WorkflowJob
import org.jenkinsci.plugins.workflow.libs.GlobalLibraries
import org.jenkinsci.plugins.workflow.libs.LibraryConfiguration
import org.junit.Before
import org.junit.Rule
import org.junit.Test
import org.jvnet.hudson.test.JenkinsRule

class MyStepTest {
    @Rule public JenkinsRule rule = new JenkinsRule()

    @Before
    void configureLibrary() {
        def retriever = LocalLibraryRetriever.implicitLibrary()
        def lib = new LibraryConfiguration('mylib', retriever)
        lib.implicit = true
        lib.defaultVersion = 'unused'
        GlobalLibraries.get().libraries = [lib]
    }

    @Test
    void myStepRuns() {
        def job = rule.createProject(WorkflowJob, 'test')
        job.definition = new CpsFlowDefinition('myStep()', true)
        rule.buildAndAssertSuccess(job)
    }
}
----
