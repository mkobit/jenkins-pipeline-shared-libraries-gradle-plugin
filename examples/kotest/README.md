# Kotest Example

This example demonstrates how to write unit and integration tests for your Jenkins pipeline shared library using [Kotest](https://kotest.io/) and Kotlin.

## Purpose
For developers who prefer Kotlin, this example shows how to configure the Gradle build to use Kotest for testing a Groovy-based Jenkins shared library. It sets up both unit and integration test suites using Kotest, demonstrating how to handle dependencies and source sets for Kotlin testing.
