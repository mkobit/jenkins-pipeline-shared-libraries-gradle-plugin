# Basic example

This example demonstrates the minimal usage of the `com.mkobit.jenkins.pipelines.shared-library` plugin.

## Purpose

It shows how to set up a basic Jenkins Pipeline shared library using the plugin, with standard source directories (`src/`, `vars/`) and basic testing.
It includes:
- Applying the plugin with minimal configuration.
- Unit tests using `JenkinsPipelineUnit`.
- Integration tests using Jenkins test harness (`JenkinsRule`).
